#  Energy Management Dashboard

An interactive web dashboard for monitoring energy consumption, waste generation, and renewable output in Gujarat, India.

## Features
- **Live Metric Telemetry**: Real-time simulated datastreams of power influx and waste levels.
- **Historical Analysis**: Deep multi-level filtering by Gujarat city, year, and month.
- **Dynamic Charting**: Radar, Matrix, Line, and Geolocation interactive modules utilizing Chart.js.
- **Premium Aesthetics**: Fully responsive Dark Mode UI featuring dynamic glassmorphism and animated components.

## Tech Stack
- HTML5
- Vanilla CSS (Glassmorphism & Dark Mode)
- Vanilla JavaScript
- Chart.js (Data Visualization)
